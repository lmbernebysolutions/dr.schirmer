# Dr. Schirmer Website - Admin Panel

## Überblick
Diese Website verwendet **Decap CMS** (früher Netlify CMS) für die Verwaltung der Inhalte. Das Admin-Panel ermöglicht es Ihnen, die Alertleiste und die aktuellen Meldungen ohne Programmierkenntnisse zu bearbeiten.

## Zugriff auf das Admin-Panel
Besuchen Sie: `https://ihre-website.vercel.app/admin/`

## Was können Sie bearbeiten?

### 1. Alertleiste
- **Sichtbarkeit**: Ein-/Ausschalten der Alertleiste
- **Text**: Der Text, der in der Alertleiste angezeigt wird
- **Automatische Aktualisierung**: Das System erkennt Änderungen automatisch

### 2. Aktuelle Meldungen
- **Titel**: Überschrift der Meldung
- **Beschreibung**: Detaillierte Beschreibung
- **Datum**: Veröffentlichungsdatum
- **Icon**: Auswahl zwischen Kalender, Herz oder Schild
- **Farbe**: Gelb oder Rot
- **Veröffentlicht**: Ein-/Ausschalten der Sichtbarkeit

## Wie funktioniert es?

1. **Bearbeitung**: Sie bearbeiten Inhalte über das Admin-Panel
2. **Git-Integration**: Änderungen werden automatisch in das Git-Repository committet
3. **Automatisches Deployment**: Vercel erkennt die Änderungen und deployed die Website neu
4. **Sofortige Verfügbarkeit**: Alle Besucher sehen die Änderungen nach wenigen Minuten

## Erste Schritte

1. Besuchen Sie das Admin-Panel
2. Melden Sie sich mit Ihrem GitHub-Account an
3. Bearbeiten Sie die gewünschten Inhalte
4. Speichern Sie die Änderungen
5. Die Website wird automatisch aktualisiert

## Technische Details

- **CMS**: Decap CMS
- **Backend**: Git Gateway
- **Hosting**: Vercel
- **Repository**: GitHub
- **Format**: JSON-Dateien

## Support

Bei Problemen wenden Sie sich an den Entwickler oder konsultieren Sie die Decap CMS-Dokumentation.
